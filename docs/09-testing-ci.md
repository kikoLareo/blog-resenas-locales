## 09. Testing y CI

### Tests

- Unitarios con Vitest (`npm run test -- --run`).
- E2E con Playwright (pendiente actualizar a la nueva UI).

### Calidad

- Type-check: `npm run type-check`
- Lint: `npm run lint`

### CI (GitHub Actions)

- Pipeline recomendado: instalar dependencias, type-check, lint, test, build.
